# nvd3-tags

A library for creating NVD3 charts from CSV data.

# Chart Options

* `type`: The type of chart. Options: `line`, `pie`, `stacked`, `bar`.

* `title`

* `width`

* `height`

* `x_start`

* `x_end`

* `x_format`

* `x_date_format`

* `y_start`

* `y_end`

* `y_format`

* `y_date_format`

* `tooltips`

* `legend`

* `clip`
